Sly is a game engine written in C++ for 3D game development.\hypertarget{index_feature_details}{}\doxysection{Features}\label{index_feature_details}
Sly provides the following features\+:
\begin{DoxyItemize}
\item Resource manager for organizing external assets such as \mbox{\hyperlink{class_image_manager}{Images}}, \mbox{\hyperlink{class_model_manager}{Models}}, \mbox{\hyperlink{class_shader_manager}{Shaders}}, and \mbox{\hyperlink{class_texture_manager}{Textures}}.
\item Base classes \mbox{\hyperlink{class_scene}{Scene}} and \mbox{\hyperlink{class_game_object}{Game\+Object}} from which to derive your own levels and entities. ~\newline

\item Built-\/in game loop to \mbox{\hyperlink{class_drawable}{Draw}} and \mbox{\hyperlink{class_updatable}{Update}} the current \mbox{\hyperlink{class_scene}{Scene}} and corresponding \mbox{\hyperlink{class_game_object}{Game\+Objects}}.
\item Collision system for \mbox{\hyperlink{class_game_object}{Game\+Objects}} to handle \mbox{\hyperlink{class_collision_volume_b_sphere}{Bounding Sphere}} collision testing. ~\newline

\item Callbacks for \mbox{\hyperlink{class_collidable}{Collision events}}, \mbox{\hyperlink{class_alarmable}{Alarm events}}, and \mbox{\hyperlink{class_inputable}{Input events}}. ~\newline
