\doxysubsection{Detailed Description}
The specifics of what an \textquotesingle{}item\textquotesingle{} is. 

A console item is composed of a name identifier at the very least along with a pointer to either a variable or a command. A console item is expandable in the sense of having a qualifier to \char`\"{}associate\char`\"{} a name + qualifier together.

The name and qualifier are stored together as a \char`\"{}description\char`\"{} internally for parsing simplicity.

Lastly, the type is determined by the input data. Data may be either a command or a data type. Command type\textquotesingle{}s are automatically populated by the console manager, whereas variable data types must be declared explicitly. ~\newline
