\doxysubsection{Detailed Description}
A cell as it is stored within the terrain. 

The structure of the cell {\itshape could} be optimized utilizing just the Y value of each min and max value within the dataset. The reason I am choosing to store the whole Vect is for quick access when creating A\+A\+BB\textquotesingle{}s for visualization and collision. ~\newline
