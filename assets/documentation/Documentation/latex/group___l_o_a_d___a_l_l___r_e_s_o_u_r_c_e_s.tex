\doxysubsection{Detailed Description}
Every game starts with loading assets. 

To begin making your first game with \mbox{\hyperlink{class_sly}{Sly}}, you will need to load resources into your game. \mbox{\hyperlink{class_sly}{Sly}} provides a number of managers to make asset management easier including a model manager, texture manager, shader manager, and image manager.
