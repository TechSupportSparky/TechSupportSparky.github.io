Sly Engine
About Sly

Sly is a game engine written in C++ for 3D game development.

Features

Sly provides the following features:

  • Resource manager for organizing external assets such as Images, Models, Shaders, and Textures.
  • Base classes Scene and GameObject from which to derive your own levels and entities.
  • Built-in game loop to Draw and Update the current Scene and corresponding GameObjects.
  • Collision system for GameObjects to handle Bounding Sphere collision testing.
  • Callbacks for Collision events, Alarm events, and Input events.
