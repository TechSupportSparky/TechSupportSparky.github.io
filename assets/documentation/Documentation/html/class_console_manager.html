static void ConsoleManager::AddCommand ( const std::string &  strName,
ConsoleCommand *  command,
int  NumArguments = 0,
std::string  iQualifier = "" 
)
inlinestatic

The process of adding a command to the console system. 

    The only requirements here are the name and command (note this must be a ConsoleCommand, inherit from this
    class).  The number of arguments and qualifier are for associating commands with objects/entities.  IE
    "map FortySeven" would be outlined as
AddCommand("map", MapCommandFactory::GetCommand(new LevelFortySeven), 1, "FortySeven");
Parameters
strNameThe name.
[in,out]commandIf non-null, the command.
NumArguments(Optional) Number of arguments.
iQualifier(Optional) Zero-based index of the qualifier.
