The place where everything lives.

The scene is the starting place to create GameObjects. A scene can be a menu, a level, or even the credits.

Getting started with a scene is easy! Pick a name for a scene and inherit from the Scene class and overload the various virtual functions derived from Scene. For example:

class Frigate; // Forward declaration
class SceneTest : public Scene {
public:
// Big four
SceneTest() {}; // Member initialization takes place in the Initialize function, not at construction.
SceneTest(const SceneTest&) = delete;
SceneTest& operator= (const SceneTest&) = delete;
~SceneTest() {}; // Cleanup of game objects takes place in SceneEnd() instead of deconstruction.
void Initialize() override; // Setup game object parameters
void SceneEnd() override; // Cleanup game object parameters
private:
Frigate *pFrigate; // The game object
}

Within the implementation of this scene, we setup the Initialize and SceneEnd functions.

void SceneTest::Initialize() {
pFrigate = new Frigate();
pFrigate.setPosition(Vect(40, 10, 40));
}
void SceneTest::SceneEnd() {
delete pFrigate;
}
