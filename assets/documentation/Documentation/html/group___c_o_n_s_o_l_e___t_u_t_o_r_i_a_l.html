Detailed Description

How to setup and initialize the developer console.

The Sly Console allows developers to execute runtime commands in real time. This ability allows for rapid prototyping of new features, heightened debugging potential, as well as user-level control over scene features.

The Sly Console functions as an interface for a user-level GameObject to interface with the Sly Engine and its game loop. With this user-level GameObject, developers may choose to have as much or as little control over the game as they choose to provide to the player with strict definitions of commands.

By default, the Sly Console comes with a handful of built-in commands that may be implemented into the developers game object. These built-in commands are as follows:

  • God Cam
  • Kill
  • Ping
  • Reload
  • Time

Since the Sly Console is expandable, there are unlimited possibilities for further commands that the developer can implement to improve their development process.
