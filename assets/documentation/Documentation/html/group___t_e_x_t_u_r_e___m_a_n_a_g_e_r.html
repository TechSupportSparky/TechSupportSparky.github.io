The texture manager provide a way for you to load textures into your games. You can either load in simple color data defined by RGBA values, or a custom texture formatted in the TARGA RAW (Truevision TGA) format AKA .tga. These textures are loaded in first with a "key" value to be referred to later and their associated texture.

Below is an example of loading both a custom texture and a simple texel texture:

#include "TextureManager.h"
Sly::LoadAllResources() {
TextureManager::Load("red", 'F', '0', '0', 'F');
TextureManager::Load("Space_Frigate", "space_frigate.tga");
}
