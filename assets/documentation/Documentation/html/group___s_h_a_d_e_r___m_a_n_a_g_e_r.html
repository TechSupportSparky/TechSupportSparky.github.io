The shader manager provides a way for you to load in shaders into your games. Each shader is associated with a "key" that can be used to refer back to the loaded shader.

Below is an example of loading in a few custom shaders.

#include "ShaderManager.h"
Sly::LoadAllResources() {
ShaderManager::Load("TextureLightRender", "textureLightRender");
ShaderManager::Load("ColorConstantRender", "colorConstantRender");
ShaderManager::Load("Sprite", "spriteRender");
}
