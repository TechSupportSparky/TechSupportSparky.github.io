The SkyBoxObject is a SkyBox that is internally maintained by the Sly Engine. An example of loading a SkyBox object is as follows:

#include "SkyBoxObjectManager.h"
Sly::LoadAllResources() {
TextureManager::Load("desertTexture", "SkyboxDesert.tga");
SkyBoxAssetManager::Load("Desert", "desertTexture", 2000, 0);
}

The above line of code will generate a SkyBox with a "desert" texture. Note, the SkyBox object merely wraps the given texture to a cube, thus the format of the image must adhere to a standard SkyBox image. The SkyBox is of size 2000 with a y offset (altitude) of 0.
